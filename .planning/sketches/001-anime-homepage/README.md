# Sketch 001: Anime Homepage Aesthetic

## Design Question
Which visual direction sets the right tone: soft feminine sakura, bouncy kawaii cards, or restrained Japanese literary?

## How to View
open .planning/sketches/001-anime-homepage/index.html

## Variants
- **A: 樱吹雪 (Sakura Minimal)** ★ Selected — Soft gradient background, falling petals, glass cards, delicate typography. Most "airy" and feminine.
- **B: 萌趣卡牌 (Moe Cards)** — Bouncing mascot, large rounded cards with tags, floating stars. Most playful and cute.
- **C: 和风軽量 (Washiki Light)** — Paper-textured warm background, numbered menu items, subtle kanji watermarks. Most refined and literary.
