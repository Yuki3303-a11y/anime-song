# Sketch Manifest

## Design Direction
Anime song quiz app ("萌豚挑战") UI redesign. Warm coral/amber palette on cream background. Target feel: clean, warm, modern anime aesthetic. Target stack: pure HTML/CSS/JS, no framework.

## Reference Points
- Japanese mobile rhythm games (Project Sekai, Bandori) — clean menu layouts
- Studio Ghibli — warm, natural color palettes
- Apple Music / Spotify — centered card layouts, big touch targets
